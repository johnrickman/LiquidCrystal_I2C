# LiquidCrystal_I2C
LiquidCrystal Arduino library for the DFRobot I2C LCD displays (based on `PCF8574` port expander)

## Wiring

Usefull for those who use bare `PCF8574`:

| PCF8574 | LCD       |
| ------- | --------- |
| P0      | RS        |
| P1      | RW        |
| P2      | E         |
| P3      | Backlight |
| P4      | D4        |
| P5      | D6        |
| P6      | D6        |
| P7      | D7        |
